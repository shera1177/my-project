# my-project# 🎯 Attendance Calculator (Flask Project)

## 📌 Description

This is a web-based Attendance Calculator developed using Flask. It calculates attendance percentage and determines eligibility based on 75% criteria.

## 🚀 Features

* Attendance percentage calculation
* Eligibility status (Eligible / Not Eligible)
* Skip class suggestion
* Required class calculation
* Modern UI with progress bar

## 🛠️ Technologies Used

* Python (Flask)
* HTML, CSS
* Jinja Templates

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
python app.py
```
